# Two-Player Tic-Tac-Toe Game

## Description
A simple Java-based two-player Tic-Tac-Toe game played on a 3×3 board.  
Players take turns entering their moves, and the program checks for a win or draw after each move.

## Features
- Two-player game (X and O)
- Console-based 3×3 board
- Win and draw detection
- Input validation

## Skills Used
- Arrays / Matrices
- Loops
- Conditional statements
- User input handling
