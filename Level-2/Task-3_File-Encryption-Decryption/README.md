# File Encryption & Decryption

## Description
A Java program that encrypts or decrypts the contents of a text file using a simple algorithm (Caesar cipher).  
The user chooses between encryption or decryption and provides the input and output file names.  
The program reads the input file, processes the content, and saves it to the output file.

## Skills Used
- File handling (reading/writing files)
- String manipulation
- Basic input/output operations
- Conditional statements
